## مرحله 1: ورود به صفحه Checkout

### فایل: `woocommerce/checkout/form-checkout.php`

### فرآیند:

#### 1.1. بررسی لاگین کاربر
```php
// خط 28-32
if (! is_user_logged_in()) {
    $url = site_url('panel?redirect=' . urlencode(home_url($wp->request) . '/?' . $_SERVER['QUERY_STRING']));
    wp_redirect($url, 301);
    exit();
}
```
**عملکرد:** اگر کاربر لاگین نباشد، به صفحه لاگین هدایت می‌شود.

---

#### 1.2. بررسی اعتبار زمان سانس
```php
// خط 34-131
if ($_GET['book'] < time()) {
    // نمایش پیام خطا: "زمان این سانس منقضی شده است"
    return;
}
```
**عملکرد:** بررسی می‌کند که زمان سانس انتخاب شده هنوز معتبر باشد.

---

#### 1.3. بررسی تداخل سانس (Race Condition)
```php
// خط 192-269
$args = [
    "single_value"  => true,
    "query"         => "SELECT * FROM `wp_zb_booking_history` WHERE `room_id` = {$product_id} AND `booking_time` LIKE {$time}",
];
$response = ez_reservation(array('type' => 'query_execution', 'data' => $args));
$row_check = (array)json_decode($response);

if (!empty($row_check)) {
    // نمایش پیام: "یکی سریع تر از تو بود و زودتر این سانس رو رزرو کرد!"
    exit();
}
```
**عملکرد:** بررسی می‌کند که سانس قبلاً رزرو نشده باشد.

**⚠️ مشکل:** این بررسی فقط در زمان لود صفحه انجام می‌شود، نه در زمان submit فرم. ممکن است بین لود صفحه و submit، سانس توسط کاربر دیگری رزرو شود.

---

#### 1.4. استخراج اطلاعات محصول
```php
// خط 138-184

```
**عملکرد:** اطلاعات محصول، زمان سانس و تعداد بلیط را از URL و سبد خرید می‌گیرد.

---

#### 1.5. نمایش فرم Checkout
- فرم اطلاعات مشتری (نام، شماره تلفن)
- فرم اطلاعات هم‌گروهی‌ها (اختیاری)

```php
// saeed-codes.php - خط 4342
add_action('woocommerce_order_status_changed', function ($order_id, $old_status, $new_status) {
    if (!in_array($new_status, ['partially-paid', 'completed-paid'], true))
        return;

        return;
```

#### 6.3. محاسبه مبالغ
```php
// خط 310-339
$pish_per_person = get_post_meta($product_id, 'pish_pardakht_per_person', true);
$pish_per_person = !empty($pish_per_person) ? $pish_per_person : 1;

$day_type = get_day_type($sans_time);
$sanses = get_sanses($product_id);

foreach ($sanses[$day_type] as $sans)
    if (date("H:i", $sans_time) == $sans['time'])
        $asli = $sans['off_price'] ?: $sans['price'];

$deposit = $pish_per_person * (int)$asli;
$item_total = $product_quantity * (int)$asli;

if ($ez_payment_type == 'partial')
    $prepaid = $deposit;
else {
    $prepaid = $item_total;
    update_post_meta($order_id, 'deposit', $deposit);
}

update_post_meta($order_id, 'prepaid', $prepaid);
$rest = (int)$item_total - $prepaid;
```

**عملکرد:**
- محاسبه مبلغ بیعانه (`deposit`)
- محاسبه مبلغ کل (`item_total`)
- محاسبه مبلغ پیش‌پرداخت (`prepaid`) بر اساس نوع پرداخت
- محاسبه مانده پرداخت (`rest`)

#### 6.4. مدیریت کیف پول و کد تخفیف
```php
// خط 342-419
$current_balance = $wldb->get_balance($user_id);
$coupon_amount = 0;
$online_paid = get_post_meta($order_id, "_order_total_2", true) ?: get_post_meta($order_id, "_order_total", true);

// محاسبه کد تخفیف
$coupons = $order->get_items('coupon');
if (!empty($coupons)) {
    foreach ($coupons as $coupon_item) {
        $code = $coupon_item->get_code();
        $coupon_amount += ez_get_coupon_discount_amount($code, $item_total);
    }
}

$wallet_share = max(0, $prepaid - ($online_paid + $coupon_amount + $user_level_discount));

if ($current_balance >= $wallet_share) {
    // شارژ کیف پول (اگر پرداخت آنلاین انجام شده)
    if ($online_paid) {
        $amount = $online_paid;
        $balance = $current_balance + $amount;
        $wldb->insert([
            'user_id' => $user_id,
            'amount' => $amount,
            'balance' => $balance,
            'description' => 'شارژ کیف پول',
            'type' => 'transaction',
        ]);
    }
    
    // کسر از کیف پول
    if ($prepaid > $coupon_amount) {
        $current_balance = $wldb->get_balance($user_id);
        $amount = ($prepaid - ($coupon_amount + $user_level_discount)) * (-1);
        $balance = $current_balance + $amount;
        $wldb->insert([
            'user_id' => $user_id,
            'amount' => $amount,
            'balance' => $balance,
            'description' => 'رزرو بازی ' . $product_title . ' - سفارش: ' . $order_id,
            'type' => 'transaction',
        ]);
    }
} else {
    // موجودی کیف پول کافی نیست - برگشت مبلغ
    // ...
    return;
}
```

**فرآیند:**
1. **شارژ کیف پول:** اگر کاربر پرداخت آنلاین انجام داده، مبلغ به کیف پول اضافه می‌شود
2. **کسر از کیف پول:** مبلغ `prepaid` (منهای کد تخفیف و تخفیف سطح) از کیف پول کسر می‌شود
3. **بررسی موجودی:** اگر موجودی کافی نباشد، مبلغ برگشت داده می‌شود

**⚠️ مشکل:** این منطق پیچیده است و ممکن است باعث confusion شود. بهتر است:
- شارژ کیف پول در یک hook جداگانه انجام شود
- کسر از کیف پول در یک hook جداگانه انجام شود

#### 6.7. بستن سانس (رزرو)
```php

    saeed_store("Prepaid is 0 : $order_id");
    return;

    (`booking_id`, `customer_id`, `wc_order_id`, `status`, `room_id`, `booking_time`, `booked_time`, `name`, `phone`, `quantity`, `level`) 
  VALUES 
    (NULL, {$user_id}, {$order_id}, 1, {$product_id}, {$sans_time}, {$now}, '{$player_name}', {$user_phone}, {$product_quantity}, {$user_level});";

$data = [
    'query' => $query,
    'single_value' => false
];

$max_attempts = 3;
$attempt = 0;
$success = false;

while ($attempt < $max_attempts && !$success) {
    ez_reservation(['type' => 'query_execution', 'data' => $data]);
    
    // بررسی موفقیت
    $check_query = "SELECT COUNT(*) AS cnt FROM `wp_zb_booking_history` 
        WHERE `customer_id` = {$user_id} 
          AND `wc_order_id` = {$order_id} 
          AND `room_id` = {$product_id} 
          AND `booked_time` = {$now}";
    
    $check_res = ez_reservation(['type' => 'query_execution', 'data' => ['query' => $check_query, 'single_value' => true]]);
    $decoded = json_decode($check_res, true);
    
    if (!empty($decoded['cnt']) && intval($decoded['cnt']) > 0) {
        $success = true;
    } else {
        $attempt++;
        usleep(200000); // 200ms delay
    }
}

if (!$success)
    saeed_store("Insert failed after {$max_attempts} attempts");

check_and_update_markting_table($order_id, $success);
```

- درج رکورد در `wp_zb_booking_history` برای رزرو سانس

- آپدیت `wp_markting` با `order_sans_time`
