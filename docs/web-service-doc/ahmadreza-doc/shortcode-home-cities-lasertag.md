# شورتکد `home-cities-lasertag`

## آدرس دقیق

- **فایل:** `wp-content/themes/escapezoom-v2/ahmadreza/shortcodes/home-cities-lasertag.php`
- **خط:** ۱ تا ۱۵۲ (کل فایل)

## امضا و پارامترها

- **نام شورتکد:** `[home-cities-lasertag]`
- **پارامتر:** ندارد (شهر پیش‌فرض و لیست شهرها ثابت است).

## کار دقیق

- پارامترهای درخواست: `source => "home_cities_lasertag"`, `params => ['city_id' => [1149]]` (کرج), `sort_type => "popular"`.
- با `ez_webservice(array('type' => 'sort_products_get', 'data' => $args))` لیست «لیزرتگ کرج» را می‌گیرد و فیلد `products` (HTML) را از JSON استخراج می‌کند.
- یک سکشن با عنوان «لیزرتگ‌های کرج»، لینک «مشاهده همه» به `home_url('/city/لیزرتگ-کرج')` و دو فیلتر چاپ می‌کند: **شهر** (تهران، کرج، اصفهان، مشهد، اردبیل، قم، گرگان با city_idهای ثابت)، **براساس** (محبوب‌ترین، پرفروش، جدیدترین).
- input مخفی با `id="cities-lasertag"` و `data-source="home_cities_lasertag"`.
- اسلایدر products-carousel با محتوای `$cities_lasertag`.
- خروجی با `ob_start()` و `return ob_get_clean()` برگردانده می‌شود.
- **وابستگی:** `ez_webservice`.

## محل استفاده در سایت

- **ثبت شورتکد:** در `ahmadreza/shortcodes/home-cities-lasertag.php`.
- **استفاده:** احتمالاً در صفحه اصلی یا صفحه لیزرتگ؛ در جستجوی فایل‌های PHP تم فراخوانی مستقیم یافت نشد. برای محل دقیق در محتوا یا قالب جستجو کنید.

## برای تغییر چه کار کنیم

1. **شهر پیش‌فرض:** مقدار `[1149]` (کرج) و عنوان و لینک «مشاهده همه» را هماهنگ کنید.
2. **لیست شهرها:** دکمه‌های شهر و city_idها را ویرایش کنید یا از تاکسونومی/آپشن بخوانید.
3. **نوع مرتب‌سازی:** مقادیر sort_type و متن دکمه‌های «براساس» را در همان فایل عوض کنید.
4. **لینک مشاهده همه:** مسیر `home_url('/city/لیزرتگ-کرج')` را با قوانین URL سایت هماهنگ کنید.

## تابع/کد مشابه در سایت

- **home-scary-cinema و city-escape-rooms:** الگوی یکسان: منبع با city_id و sort_type، سکشن با عنوان + فیلتر شهر + فیلتر براساس + اسلایدر. تفاوت در منبع، شهر پیش‌فرض و لیست شهرها.
- **ادغام:** تابعی مثل `ez_render_city_products_section($source, $default_city_id, $cities_list, $title, $view_all_slug)` و استفاده هر شورتکد از آن با پارامترهای مخصوص خود؛ قالب و منطق فیلتر یکجا باشد.

## بهینه‌سازی

1. **چک پاسخ webservice:** قبل از چاپ `$cities_lasertag` معتبر بودن پاسخ را چک کنید.
2. **شهرها از دیتابیس/آپشن:** لیست شهرها را از terms یا آپشن بخوانید تا بدون تغییر کد قابل مدیریت باشند.
3. **یکسان‌سازی با home-scary-cinema و city-escape-rooms:** استخراج قالب مشترک (سکشن + فیلتر شهر + براساس + اسلایدر) در یک تمپلیت/تابع.
4. **آیدی یکتا برای SVG:** در صورت استفاده چند بار شورتکد در یک صفحه، idهای clipPath در SVG را یکتا کنید.
