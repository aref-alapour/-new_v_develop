# تابع `jstrftime`

## آدرس دقیق

- **فایل:** `wp-content/themes/escapezoom-v2/ahmadreza/jdate.php`
- **خط:** حدود ۲۹۲ تا ۵۹۰

## امضا

```php
function jstrftime($format, $timestamp = "", $none = "", $time_zone = "Asia/Tehran", $tr_num = "fa")
```

- **$format:** سینتکس strftime (مثلاً %Y, %m, %d, %H:%M).

## کار دقیق

- مشابه jdate با **سینتکس strftime**؛ تاریخ میلادی را به شمسی تبدیل و طبق placeholderها برمی‌گرداند.

## محل استفاده در سایت

- **inc/saeed-codes.php** حدود خط ۵۶۶۰: `jstrftime('%H:%M روز %Y/%m/%e', $row['booking_time'])` برای زمان رزرو در جدول/لیست.
- بقیه تم از `jdate` استفاده می‌کنند.

## برای تغییر چه کار کنیم

1. در saeed-codes همان خط را ویرایش کنید.
2. می‌توان همان خروجی را با ترکیب `jdate('H:i', $ts)` و `jdate('Y/m/j', $ts)` و متن «روز» ساخت و jstrftime را حذف کرد.

## تابع/کد مشابه در سایت

- **jdate:** همان منطق با سینتکس دیگر؛ در تم غالباً jdate استفاده شده.
- **ادغام:** آن یک فراخوانی را می‌توان به jdate با فرمت مناسب تبدیل کرد.

## بهینه‌سازی

1. یکسان‌سازی با jdate برای کاهش دو مسیر فرمت.
2. در صورت عدم استفاده در جاهای دیگر، فقط همان یک فراخوانی را عوض کنید و jstrftime را برای سازگاری نگه دارید یا حذف کنید.
