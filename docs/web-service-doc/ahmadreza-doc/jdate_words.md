# تابع `jdate_words`

## آدرس دقیق

- **فایل:** `wp-content/themes/escapezoom-v2/ahmadreza/jdate.php`
- **خط:** حدود ۷۰۸ تا ۹۲۸

## امضا

```php
function jdate_words($array, $mod = "")
```

- **$array:** آرایه نوع→عدد؛ مثلاً ["mm" => $j_m] برای نام ماه، ["rh" => $date[7]] برای نام روز هفته.
- **$mod:** اگر رشته باشد خروجی با این جداکننده به هم وصل می‌شود؛ وگرنه آرایه برمی‌گردد.

## کار دقیق

- اعداد را به **متن فارسی** تبدیل می‌کند: نام ماه (فروردین، اردیبهشت، ...)، نام روز (یکشنبه، ...)، روز ماه (یک، دو، ...)، سال به حروف، فصل و ...
- نوع‌ها: ss, mm, rr, rh, sh, mb, ff, km, kh و غیره.

## محل استفاده در سایت

- فقط **داخل jdate.php** توسط jdate و jstrftime برای تولید نام ماه، روز هفته و ...
- در تمپلیت‌ها فراخوانی مستقیم یافت نشد؛ استفاده غیرمستقیم از طریق jdate('F', $ts) یا jdate('l', $ts).

## برای تغییر چه کار کنیم

- تغییر متن‌ها (نام ماه‌ها و ...) فقط با ویرایش آرایه‌های داخل jdate_words در jdate.php.
- در صورت نیاز به نام ماه/روز در جای دیگر: `jdate_words(['mm' => $month], '')`.

## تابع/کد مشابه در سایت

- jdate خروجی نهایی تاریخ با حروف را از این تابع می‌گیرد.

## بهینه‌سازی

- بدون نیاز به تغییر متن یا منطق؛ بخش ضروری کتابخانه jdate است.
