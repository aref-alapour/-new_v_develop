# تابع `jmktime`

## آدرس دقیق

- **فایل:** `wp-content/themes/escapezoom-v2/ahmadreza/jdate.php`
- **خط:** حدود ۵۹۴ تا ۶۵۵

## امضا

```php
function jmktime($h = "", $m = "", $s = "", $jm = "", $jd = "", $jy = "", $none = "", $timezone = "Asia/Tehran")
```

## کار دقیق

- معادل **mktime** برای تاریخ شمسی: اجزای زمان و تاریخ شمسی را می‌گیرد و **Unix timestamp** برمی‌گرداند.
- از `jalali_to_gregorian` و `mktime` استفاده می‌کند.

## محل استفاده در سایت

- فقط **داخل jdate.php** توسط توابع دیگر (مثل jgetdate) استفاده می‌شود.
- در تمپلیت‌ها فراخوانی مستقیم `jmktime` یافت نشد.

## برای تغییر چه کار کنیم

- اگر جایی در کد سفارشی از jmktime استفاده می‌کنید، پارامترها را طبق مستندات jdate تنظیم کنید.
- تغییر منطق فقط در jdate.php و با توجه به سازگاری با بقیه jdate.

## تابع/کد مشابه در سایت

- PHP: `mktime()` برای میلادی.
- jdate.php: `jalali_to_gregorian` داخل jmktime استفاده شده.

## بهینه‌سازی

- بدون استفاده مستقیم در تم؛ به‌عنوان بخشی از کتابخانه jdate نگه داشته می‌شود.
