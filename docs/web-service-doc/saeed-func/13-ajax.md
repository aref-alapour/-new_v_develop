# AJAX سایت

## ez_site_ajax_handler_callback

- **تابع در saeed-codes.php:** `ez_site_ajax_handler_callback()`
- **جایگزین:** تابع اصلی است؛ هندلر مرکزی AJAX برای فرانت و اپ.
- **جایی که صدا زده می‌شود:** اکشن‌های `wp_ajax_ez_site_ajax_handler` و `wp_ajax_nopriv_ez_site_ajax_handler`.
- **کارایی:** بر اساس پارامتر ارسالی (مثلاً action یا type)، درخواست را به بلوک مناسب هدایت می‌کند (جستجو، رزرو، سانس، کوپن، تیکت و غیره) و خروجی JSON یا HTML برمی‌گرداند.
- **بهینه‌سازی:** (۱) بلوک‌های بزرگ را به توابع جدا با نام ثابت ببر و با یک switch یا map بر اساس action صدا بزن تا تابع اصلی کوتاه و خوانا بماند. (۲) اعتبارسنجی nonce و دسترسی (نقش/کاربر) را در ابتدا انجام بده. (۳) خطاها را با کد و پیام یکسان برگردان تا فرانت یکسان هندل کند. (۴) برای actionهای سنگین، پاسخ فوری بده و کار اصلی را در صف انجام بده.
