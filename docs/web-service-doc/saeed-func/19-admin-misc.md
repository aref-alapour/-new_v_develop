# ادمین متفرقه

## remove_footer_admin
- **تابع در saeed-codes.php:** `remove_footer_admin()` — فیلتر `admin_footer_text`
- **جایگزین:** تابع اصلی است.
- **کارایی:** متن فوتر ادمین وردپرس را خالی یا سفارشی می‌کند.
- **بهینه‌سازی:** در صورت نیاز به متن سفارشی، از __() برای ترجمه استفاده کن؛ یا خالی برگردان.

## withdrawal_owner_profile_fields / save_withdrawal_owner_profile_fields
- **کارایی:** در پروفایل کاربر (مالک) فیلدهای مربوط به برداشت (شماره شبا، نام بانک و غیره) را نمایش و ذخیره می‌کنند.
- **بهینه‌سازی:** نام متاها را در ثابت تعریف کن؛ اعتبار شبا و فیلدها را چک کن؛ فقط برای نقش مالک نمایش بده.

## admin_hide_items_via_css
- **تابع در saeed-codes.php:** `admin_hide_items_via_css()` — اکشن `admin_head`
- **کارایی:** با CSS برخی آیتم‌های منو یا صفحهٔ ادمین را مخفی می‌کند.
- **بهینه‌سازی:** سلکتورها را در یک جا نگه دار؛ ترجیحاً به‌جای مخفی کردن، با remove_submenu_page منو را حذف کن تا دسترسی واقعاً محدود شود.
