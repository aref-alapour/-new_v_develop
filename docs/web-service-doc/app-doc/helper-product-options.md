# helper/product-options.php

**مسیر:** `app/functions/helper/product-options.php`  
**بارگذاری:** از طریق `app/functions/init.php`.

---

## نقش کلی

نمایش آپشن‌های امکانات محصول (جای پارک، کافه، سوپرایز تولد، دسترسی، اتاق انتظار، سرویس بهداشتی، تهویه) به صورت کارت با آیکون SVG و عنوان فارسی.

---

## تابع `printOptions($options)`

- **ورودی:** `$options` — آرایه‌ای از کلیدهای آپشن (مثلاً `['easy_park','cafe']`).
- **خروجی:** ندارد؛ مستقیم HTML echo می‌کند.

**آپشن‌های تعریف‌شده در فایل:**  
`easy_park`, `cafe`, `surprise`, `transport`, `waiting_room`, `wc`, `ventilation` — هر کدام `title` و `icon` (رشته SVG) دارند.

برای هر عنصر `$options` اگر در `$value_options` باشد یک div با کلاس‌های Tailwind و آیکون و عنوان چاپ می‌شود؛ وگرنه همان div با متن «آیکون شناسایی نشد: {option}».

**کجا استفاده می‌شود:** در قالب، معمولاً در صفحه محصول یا لیست محصولات، جایی که امکانات بازی/محصول نمایش داده می‌شود.

**تغییر:** برای آپشن جدید، یک کلید جدید به آرایه `$value_options` با `title` و `icon` اضافه کنید. برای تغییر استایل، کلاس‌های داخل echo را ویرایش کنید.

**بهینه‌سازی:** می‌توان به‌جای echo مستقیم، خروجی را در متغیر جمع کرد و return کرد تا قالب کنترل بهتری روی خروجی داشته باشد؛ یا از `get_template_part` / بلوک برای جدا کردن HTML استفاده شود.
