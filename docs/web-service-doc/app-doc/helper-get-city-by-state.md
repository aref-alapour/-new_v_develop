# helper/get-city-by-state.php

**مسیر:** `wp-content/themes/escapezoom-v2/app/functions/helper/get-city-by-state.php`  
**بارگذاری:** در `app/functions/init.php` **include نمی‌شود**. فقط با `require`/`include` از جایی که به شهرها بر اساس استان نیاز است لود می‌شود.

---

## خلاصه

فایل **داده‌ای**: آرایه‌ای با ساختار **کد استان => لیست نام شهرها** برمی‌گرداند. هیچ تابعی تعریف نمی‌کند؛ فقط یک آرایه با `return` دارد. کلیدها کدهای استان (مثلاً KHZ, THR, ILM, BHR, ADL, ESF, YZD, KRH, KRN, …) و مقدار هر کلید آرایه‌ای از نام شهرهای آن استان است.

---

## محتوا

**خطوط تقریبی:** ۱–۱۴۰۷.

ساختار کلی:
```php
return [
    'KHZ' => [ 'آبادان', 'آبژدان', ... ],
    'THR' => [ 'آبسرد', 'آبعلی', ... ],
    'ESF' => [ 'آران و بیدگل', ... ],
    // ...
];
```

کدهای استان موجود شامل: KHZ, THR, ILM, BHR, ADL, ESF, YZD, KRH, KRN, HDN, GZN, ZJN, LRS, ABZ, EAZ, WAZ, CHB, SKH, RKH, NKH, SMN, FRS, QHM, KRD, KBD, GLS, GIL, MZN, MKZ, HRZ, SBN و غیره.

---

## استفاده در سایت

- در فرم‌های آدرس یا فیلتر بر اساس استان: بعد از انتخاب استان، لیست شهرها از این آرایه با کلید کد استان گرفته می‌شود.
- در پنل یا وب‌سرویس برای پر کردن dropdown شهر بر اساس استان.

---

## تفاوت با cities_type.php

- **cities_type.php:** بر اساس **نوع محصول** (اتاق فرار، سینما ترس، …) لیست شهرها با `city_id` و `city_name` برمی‌گرداند.
- **get-city-by-state.php:** بر اساس **کد استان** لیست **نام شهرها** (بدون id) برمی‌گرداند.

هر دو منبع داده «شهر» هستند ولی یکی برای فیلتر محصول و دیگری برای آدرس/استان است.

---

## نحوه تغییر

- **اضافه/ویرایش شهر:** در آرایه مربوط به هر کلید استان، المان‌های رشته‌ای را ویرایش کنید.
- **اضافه استان جدید:** یک کلید جدید (کد استان) با آرایه نام شهرها اضافه کنید.
