# مستندات توابع web-service.php

هر فایل MD در این پوشه مربوط به **یک تابع** فایل `web-service/web-service.php` است.

در هر فایل آمده است:
- **تابع در web-service.php:** نام و امضای تابع
- **کارایی:** دقیقاً چه کاری انجام می‌دهد و کجا صدا زده می‌شود
- **جایگزینی:** چطور می‌شود این تابع را جایگزین یا حذف کرد
- **بهینه‌سازی:** پیشنهاد برای تمیزتر/سریع‌تر/قابل نگهداری‌تر شدن
